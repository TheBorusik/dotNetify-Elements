### How To Run This Repo

Get [.NET Core 2.1 SDK](https://www.microsoft.com/net/download/dotnet-core/sdk-2.1.300).  
On a VS Code terminal:

```
cd DevApp
npm i 
dotnet build
dotnet watch run
```
